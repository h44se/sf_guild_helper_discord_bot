- Added to checkfight.js function to check multiple fights in one .har.

`Anzahl erkannter Kämpfe: 1.`

`Kampf #1 gegen GUILDTESTA:`

```txt
Anzahl erkannter Kämpfe: 1

Kampf #1 gegen GUILD:
```
